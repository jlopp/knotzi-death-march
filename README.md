Knotzi Death March Countdown
=====================================

What is this project's purpose?
----------------

This is a simple countdown timer to Bitcoin block height 961,632 which is the first block at which BIP-110 nodes enforce mandatory signaling.

In other words, BIP-110 nodes will reject any new blocks that DO NOT signal for BIP-110.

At time of writing, less than 0.1% of blocks are signaling for BIP-110.

As such, in all likelihood, block 961,632 will be the height at which BIP-110 nodes stop following the Bitcoin blockchain and are effectively forked off the network.

Whether or not an alternative chain fork emerges after this point will depend upon if any further BIP-110 signalling blocks are mined.

Even if blocks are found, it's unlikely that such a chain fork will persist for long because it would take years to get to the next difficulty adjustment and blocks would be few and far between.

The most likely outcome is that BIP-110 nodes are essentially frozen and stop being useful as of block height 961,632.
