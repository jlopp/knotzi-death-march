Knotzi Death March Countdown

This countdown timer tracks how long it will be until people running BIP-110 nodes will be forked off the Bitcoin network, most likely to a dead network that produces no new blocks.

For background info, see A Layman's Guide to BIP-110 and It's Knot a Serious Project .

To see the status of BIP-110 miner signaling, check this chart .

To compare current block heights reported by different node implementations, check bitnod.es.

The Knotzi Death March Countdown is 100% open source and available on GitHub .
